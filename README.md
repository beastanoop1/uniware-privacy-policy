# Privacy Policy for Uniware Chrome Extension

Last updated: July 6, 2026

Uniware is a Chrome extension that helps filter and organize the Unicommerce picklist by service type, making it easier for warehouse staff to view and process orders.

## Data Collection

This extension does not collect, store, transmit, or share any personal data, browsing data, or user activity. All filtering happens locally within your browser on the Unicommerce picklist page you are already viewing. No information is sent to any external server, third party, or the developer.

## Permissions

The extension requests only the minimum browser permissions (activeTab, tabs, scripting, and host access to the Unicommerce domain) required to read and filter the picklist content displayed on the page. These permissions are used solely to provide the filtering feature and for no other purpose.

## Third-Party Sharing

We do not sell, rent, or share any data with third parties, since no data is collected in the first place.

## Changes to This Policy

If this policy changes in the future, updates will be posted on this page.

## Contact

For questions about this privacy policy, please open an issue on this repository.
